L'application a été développée à l'aide du navigateur google chrome.

L'application a été testée sur un équipement android Samsung Galaxy S4.

L'exécution a été rencontré des problèmes d'exécution sur l'émulateur de Titanium.

L'application charge un premier sondage à partir d'un fichier JSON.

L'utilisation du swipe horizontal permet de faire défiler les questions.

Après avoir répondu aux questions du sondage et l'avoir validé, l'application enregistre les résultats dans un fichier JSON.

Ensuite, il est possible de choisir un autre sondage parmi les sondages proposés qui sont eux mêmes chargés à partir de fichiers JSON.

Il est également possible de visualiser les résultats en cliquant sur l'onglet résultat.

L'utilisation du swipe vertical permet de visualiser l'intégralité du fichier résultat.

L'onglet aide (bouée) permet de visualiser le nom des développeurs.

L'execution de l'application writeJSONFile permet de réinitialiser les fichiers résultat à 0.
